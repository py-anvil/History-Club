# 关于我们

> 在这里介绍社团的基本信息。

## 社团简介

湄轩历史社 是福建省莆田一中的历史社团，是一个致力于了解历史，讨论历史并得出结论的社团。

## 社团概况

- **社团名称**: 湄轩历史社
- **口号**: 历史即鉴
- **社团历史**: 社团历史悠久，有热衷历史与热爱社团活动的社友

## 社活内容

- 讨论历史等
- 欢迎新人提出合理的新社活内容
